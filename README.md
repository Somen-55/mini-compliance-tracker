# 📊 Mini Compliance Tracker

A full-stack web application designed to manage compliance tasks across multiple clients.  
Built using the MERN stack (MongoDB, Express, React, Node.js) and deployed on Render.

---

## 🚀 Live Demo

- 🌐 Frontend: https://mini-compliance-tracker-7f6y.onrender.com/  
- 🔧 Backend: https://mini-compliance-tracker-backend-2ywc.onrender.com  

---

## ✨ Features

- Add and manage clients  
- Create, update, and delete tasks  
- Client-specific task tracking  
- Task status management  
- Real-time updates without page refresh  
- Clean and responsive UI  

---

## 🛠️ Tech Stack

Frontend: React (Vite), Axios, Bootstrap  
Backend: Node.js, Express.js, MongoDB (Mongoose), CORS, dotenv  
Deployment: Render, MongoDB Atlas
